# SecureDrive: Detecção de Fadiga e Assistência de Rota

Este repositório contém o código-fonte da página de apresentação do projeto **SecureDrive**, um sistema inteligente de segurança veicular apresentado na 8ª edição do IFCIÊNCIA.

## Qual o problema abordado?

O sono e a fadiga são a terceira maior causa de riscos comportamentais nas estradas, contribuindo para cerca de **60% dos acidentes de trânsito**. Diante disso, o projeto SecureDrive foi criado não apenas para alertar o motorista, mas para oferecer uma solução proativa de segurança.

## Sobre o Projeto:

O SecureDrive é um sistema que utiliza uma câmera e algoritmos de Machine Learning (Aprendizado de Máquina) para monitorar o motorista em tempo real.

### Funcionalidades Principais:

* **Detecção de Fadiga:** O software analisa pontos específicos dos olhos e boca do motorista, contando piscadas e bocejos em intervalos de 20 segundos para identificar os primeiros sinais de sonolência.
* **Rota para Descanso (O Diferencial):** Ao detectar fadiga, o sistema não se limita a um alarme. Ele se integra à **API do Google Maps** para localizar automaticamente estabelecimentos seguros próximos (como postos de gasolina ou hotéis) e sugere uma nova rota para que o motorista possa descansar.

### Aplicação Inesperada: Saúde

Durante o desenvolvimento, a equipe percebeu que a mesma tecnologia de *eye-tracking* (rastreamento ocular) de alta precisão usada para detectar a sonolência tem um potencial surpreendente: auxiliar na **identificação precoce do espectro autista em crianças**, analisando padrões de olhar que muitas vezes não são percebidos pela observação humana comum.
